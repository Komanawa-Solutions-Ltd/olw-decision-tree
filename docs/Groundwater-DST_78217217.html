OLW Monitoring Design Framework : Groundwater DST

Created by Zeb Etheridge, last modified by Olivier Ausseil on Mar 31, 2021

Overview

The Groundwater Decision Support Tool comprises:

  1. A flow chart to assist with method selection

  2. A series of methods to be applied in a given set of circumstances to assist with monitoring design

  3. Information pages which guide and assist with use of the DST (currently incomplete)

  4. A scenario-based case study to demonstrate usage of the DST
