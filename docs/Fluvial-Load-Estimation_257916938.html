OLW Monitoring Design Framework : Fluvial Load Estimation

Created by Olivier Ausseil on Jan 04, 2021

  • Suspended Sediment Load Estimation: National Environmental Monitoring Standard (NEMS) http://www.nems.org.nz/documents/suspended-sediment/

  • Averaging method

  • Beale Estimator ratio
