OLW Monitoring Design Framework : Statistical analysis module

Created by Zeb Etheridge, last modified by Amy Whitehead on Mar 30, 2021

Overview

The statistical analysis module provides a broad overview of statistical techniques that can be used to design monitoring networks. The guidance provided typically includes a brief description of available methodologies, with links to key references. In addition, we provide worked examples linked to the specific case studies related to the riparian and groundwater DSTs. The primary questions covered include:

  • How many samples or sites do I need? Will my network be able to detect the changes expected due to mitigation in a reasonable time period?

    • Power analysis

  • Where are monitoring sites best placed within the landscape to detect change?

    • Monitoring Network Design

  • Will my monitoring network be representative of the changes expected across the landscape?

    • Spatial representation analysis

  • What analysis options are available for detecting changes over time?

    • Temporal Trend Assessments

We note that the guidance provided is most relevant to monitoring using discrete sampling (e.g., monthly samples) rather than continuous monitoring.
